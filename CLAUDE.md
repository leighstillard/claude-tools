# claude-tools

Setup script for Claude Code developer tooling. This repo complements [claude.md-boilerplate](https://github.com/leighstillard/claude.md-boilerplate) — use that repo for project-level engineering standards.

## What This Repo Does

Installs and configures the recommended developer tools from the boilerplate's [Recommended Developer Tools](https://github.com/leighstillard/claude.md-boilerplate#recommended-developer-tools) section:

- **RTK** (Rust Token Killer) — CLI proxy for 60-90% token savings
- **LSP servers** — semantic code understanding (pyright, gopls, typescript-language-server)
- **Claude Code plugins** — superpowers, code-review, security-guidance

## Verification

When asked to "verify developer tools" or on first interaction after setup, read `docs/standards/devtools-verification.md` and follow its steps.

## Standards

This repo follows the engineering standards defined in [claude.md-boilerplate](https://github.com/leighstillard/claude.md-boilerplate). Refer to that repo's `CLAUDE.md` and `docs/standards/` for full project standards.
